# Shared / cross-repo ADRs (synced copies)

Every `.md` file in this folder except this README is a **synced copy** generated by `npm run adr:sync` (run from `client/`). The canonical versions are authored in the backend repo under `packages/contracts/adr/shared/` and ship inside the `@umgccapstone/contracts` package under `/adr/shared/`.

- **Do not edit these files here.** Propose changes via PR against the backend repo's `packages/contracts/adr/shared/`, then bump the package and re-run the sync.
- CI runs the sync and fails the build if the committed copies drift from the installed package version.
- Until `@umgccapstone/contracts` is published and installed (backend T-0), the sync is a no-op and the committed copies reflect the canonical content at the time of the last manual sync.
